# Table Options

A table can be configured by passing in a configuration object to the `options` prop on the
[`Tabulus`](@todo) comonent, under the `options` key of the [`useTabulus`](@todo) hook.

## Available Options

### Layout

| Option | Value Types                                | Default Value | Description                                                   |
| ------ | ------------------------------------------ | ------------- | ------------------------------------------------------------- |
| align  | <code>'left' \| 'center' \| 'right'</code> | `'center'`    | Whether the column should be displayed in the rendered table. |

### Columns

| Option         | Value Types | Default Value                            | Description                                                                                    |
| -------------- | ----------- | ---------------------------------------- | ---------------------------------------------------------------------------------------------- |
| columnDefaults | `object`    | [See Column Options]('../columnOptions') | Default options to use for columns. These can be overridden in a [column](@todo)'s definition. |
