---
title: CellComponent
---

The `CellComponent` represents an individual cell in the table. This can either be in the header, or
as part of the table body.

By default, the `Cell` React component offers all of the functionality provided by `tabulus`,
however you may override the component with a custom one. If you do this, you will need to ensure
that certain props can be passed in to the component.
