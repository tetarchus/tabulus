# Styling Tabulus

Tabulus offers multiple different methods for styling your tables. The simplest method is to just
pick a pre-made theme by passing in a string option to the `theme`
[option]('../config/tableOptions.md').

### CSS Classes

| Class             | Element Description                                                            |
| ----------------- | ------------------------------------------------------------------------------ |
| tabulus           | The main table container                                                       |
| tabulus-head      | The header element of the table                                                |
| tabulus-head-row  | A row within the header element                                                |
| tabulus-body      | The main table body, containing rows that are not part of the header or footer |
| tabulus-row       | A row within the table                                                         |
| tabulus-row-even  | An evenly numbered row - to allow for striping                                 |
| tabulus-row-odd   | An oddly numbered row - to allow for striping                                  |
| tabulus-col       | A column header cell                                                           |
| tabulus-col-title | The title wrapper for a column                                                 |
