The simplest way of styling a `tabulus` table is using a theme. You can use one of our built-in
themes, modify one of the built-in themes, or make a completely custom one.

@todo

## Theme Options

When defining a theme in the options you can do this in a number of ways.

### 1. Built-In Theme Name

```ts
const options = {
  theme: 'standard',
};
```

### 2. Theme Extension

```ts
const options = {
  theme: {
    baseTheme: 'standard',
    borders: 'none',
  },
};
```
