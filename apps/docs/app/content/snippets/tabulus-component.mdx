## 1. Tabulus Component

The simplest method for creating a table is with the `<Tabulus />` component.

To create a table, simply pass in the [column definitions](@todo), [data](@todo) and any
[options](@todo) you want to pass to the table.

```ts
import {useState} from 'react';
import {Tabulus} from 'tabulus';

const App = () => {
  const [columns, setColumns] = useState([]);
  const [data, setData] = useState([]);

  return <Tabulus columns={columns} data={data} />
}

```
